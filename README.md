# ESTRELLA
The website provides final products for the NSF ESTRELLA project (NSF Grant No. 1949230). 

## [6th-grade lessons for visualizing fraction operations using Python and digital videos](https://github.com/pattichis/fractions)
The final project includes the creation of a video for visualizing multiplying and dividing fractions.

# 7th-grade lessons for visualizing linear relationships using Python and digital videos
[GitHub for visualizing linear relationships with Python-generated videos](https://github.com/pattichis/GraphSpeeds2/blob/main/README.md)
The final project includes making a race video with realistic speeds using custom characters. 

# Visualize line geometry using Python
[GitHub for visualizing line geometry using Python](https://github.com/pattichis/lineart)
The final project includes building a house using lines and points.

# Visualize exponential functions using Python and videos: Make a flower and a face using exponentials
[GitHub for visualizing exponential functions ](https://github.com/pattichis/graphfuns2)
The activities include two projects. The first project requires students to make a flower using exponential functions.
The second project requires students to make a face using exponential functions.
